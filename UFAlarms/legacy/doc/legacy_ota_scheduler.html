Legacy OTA scheduler

Handle OTA jobs received from datadealer.

It creates a worker queue to handle firmware update, it will spwan_link a legacy_ota_worker for each node and listen for Exist messages and publish OTA updates.

Authors: Pierre Defebvre (pierre.defebvre@verizon.com).
